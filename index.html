The Setup

From zero to colony in minutes

1
✦ Free

Get your Gemini API key

Google gives you free access to Gemini 2.5 Flash — the model powering your colony's main intelligence. No credit card required. 30 seconds to get the key.

Open Google AI Studio →

Copy the key that starts with AIza... — you'll need it in Step 4.

2
✦ ~$5

Get your Claude API key

Anthropic's Claude powers your colony's memory (SOMA), synthesis, and deep reasoning. A $5 credit gets you 2–3 months of typical colony usage.

Open Anthropic Console →

Create a key that starts with sk-ant-... — keep it secret, never share it.

3
✦ Install

Install the colony stack

The colony runs locally on your machine using Node.js. Clone the repo and install dependencies — this is a one-time setup.

Requires: Node.js 18+ and Git. Install Node.js →

# Clone and install
git clone https://github.com/jafstar/syncagent-v2.git colony
cd colony
npm install
4
✦ Launch

Configure and launch

Create your environment file with your keys, then start the colony. All your agents wake up automatically.

# Create your .env file
ANTHROPIC_API_KEY=sk-ant-your-key-here
GEMINI_API_KEY=AIza-your-key-here

# Launch the colony
node colony.mjs

Your colony is live. Open colony.aint.farm to start directing it.
